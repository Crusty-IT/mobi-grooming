# Changelog

Wszystkie istotne zmiany w tym projekcie będą dokumentowane w tym pliku. Format oparty jest na [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-12-30
### Added
- Utworzenie pełnej dokumentacji projektu (`README.md`, `docs/`).
- Konfiguracja Netlify CMS dla ogłoszeń i kalendarza.
- Implementacja sekcji: Dashboard, Galeria, Opinie, Certyfikaty, Informacje.
- System dynamicznego ładowania ogłoszeń z plików Markdown.
- Kalendarz niedostępności ładowany z pliku JSON.
- Integracja z Netlify Identity.
- Stylizacja za pomocą Tailwind CSS 4.
- Ikony z biblioteki Lucide React.
- Konfiguracja eksportu statycznego w Next.js.

## [0.1.0] - 2025-10-15
### Added
- Inicjalizacja projektu Next.js 15.
- Podstawowa struktura komponentów.
- Konfiguracja Tailwind CSS.
